@[devblog.sanyamkapoor.com](http://devblog.sanyamkapoor.com)
and [activatedgeek.github.io](http://activatedgeek.github.io).

### Usage

To build on a local development server for testing
```
$ make debug
```

To generate and deploy to Github Pages
```
$ make publish
```
